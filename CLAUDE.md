# CLAUDE.md

## Project goal
Build a premium interactive website about iconic vintage game consoles.

## Tech stack
- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion
- shadcn/ui allowed when useful

## Design direction
- Modern premium look
- Retro-futuristic vibe
- Dark theme
- Subtle CRT / scanline / grain effects only
- No kitsch, no childish arcade overload
- Smooth animations, never cheap
- Strong editorial / collector museum feel

## UX rules
- Mobile-first and fully responsive
- Fast loading
- Clear hierarchy
- Accessible contrast
- Elegant hover / scroll interactions
- Avoid over-animation

## Code rules
- Reusable components
- Clean folder structure
- Strong typing
- No dead code
- No giant monolithic components
- Prefer data-driven rendering from structured datasets

## Content rules
Each console entry should support:
- release dates by region
- manufacturer
- generation
- iconic games
- emotional score
- rarity
- second-hand link
- anecdote
- simplified specs

## Workflow
Before coding:
1. Inspect project structure
2. Propose implementation plan
3. Then implement step by step

After each major change:
- run lint
- run build if possible
- report what changed

## Output style
Be concise, practical, and opinionated.
Make strong design decisions when needed.
